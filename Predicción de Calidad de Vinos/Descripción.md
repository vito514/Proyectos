# Predicción de calidad de vinos

El objetivo de este proyecto de ciencia de datos es predecir la calidad de los vinos usando el dataset "Wine Quality". Este contiene dos conjuntos de datos relacionados con muestras de vino "Vinho Verde" en sus variantes tinto y blanco, provenientes del norte de Portugal.

Cada vino fue evaluado sensorialmente por expertos, quienes le asignaron una puntuación de calidad en una escala de 0 a 10.

Este dataset se presta para tareas de clasificación y regresión y es útil para analizar qué características influyen en la calidad del vino.
